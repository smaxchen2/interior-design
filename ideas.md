# 北歐風室內設計 3D 互動展示 — 設計構思

## 目標
基於用戶提供的平面圖（隔間完全固定），使用 Three.js 100% 精確還原所有牆體、門窗位置，並以現代簡約北歐風進行室內設計優化（餐桌加大），提供互動式 3D 瀏覽。

## 方案一：建築藍圖數位化（Architectural Blueprint Digital）

**設計運動**：數位建築主義（Digital Architecturalism）

**核心原則**：
1. 精確的建築比例與尺寸還原
2. 清晰的空間層次與動線標示
3. 專業的材質與光影表現

**色彩哲學**：以建築圖紙的冷灰藍為基調，搭配暖木色的北歐元素。背景使用深藍灰（#1a2332），UI 控件使用白色與金色點綴，營造專業但溫暖的感受。

**佈局範式**：左側為全螢幕 3D 視窗，右側為可收合的控制面板。底部為空間導航條，可快速切換不同房間視角。

**簽名元素**：
1. 精確的建築線條動畫（牆體逐步繪製）
2. 測量標註浮動顯示（hover 時顯示尺寸）

**互動哲學**：如同在專業 CAD 軟體中瀏覽，但更直覺。滑鼠拖曳旋轉、滾輪縮放、雙擊聚焦房間。

**動畫**：牆體從平面圖「生長」為 3D 的過渡動畫，家具淡入效果。

**字體系統**：Noto Sans TC（中文）+ DM Sans（英文數字），標題粗體，數據使用等寬字體。

## 方案二：北歐展廳體驗（Nordic Showroom Experience）

**設計運動**：斯堪地納維亞極簡主義（Scandinavian Minimalism）

**核心原則**：
1. 大量留白與呼吸感
2. 自然材質的真實質感
3. 功能性優先的介面設計

**色彩哲學**：整體 UI 使用溫暖的米白色（#faf8f5）為背景，搭配淺木色（#d4b896）作為強調色，深炭灰（#2d2d2d）作為文字色。這模擬了走進一間北歐風格展示間的感受。

**佈局範式**：全螢幕沉浸式 3D 視窗為主體，頂部為極簡導航列，左下角為房間選擇器（圓形縮圖），右側為可展開的設計說明面板。

**簽名元素**：
1. 木紋質感的 UI 邊框與按鈕
2. 柔和的環境光漸變效果

**互動哲學**：如同在 IKEA 展間中漫步，點擊房間自動飛行過去，家具可點擊查看詳細資訊卡片。

**動畫**：相機平滑飛行過渡、家具資訊卡片從底部滑入、光影隨時間緩慢變化。

**字體系統**：Noto Serif TC（中文標題）+ Outfit（英文），營造優雅但現代的閱讀體驗。

## 方案三：互動式設計提案（Interactive Design Proposal）

**設計運動**：功能主義互動設計（Functional Interactive Design）

**核心原則**：
1. 2D 平面圖與 3D 視圖的無縫切換
2. 原圖與優化方案的即時對比
3. 清晰的設計決策說明

**色彩哲學**：UI 使用中性的暖灰白（#f5f3f0），3D 場景中忠實呈現北歐風配色。控制元素使用莫蘭迪灰藍（#7d8fa3）作為主色調，與室內設計風格呼應。

**佈局範式**：上方為分頁式導航（總覽 / 客廳 / 餐廚 / 主臥 / 次臥 / 陽台），每個分頁包含左側 3D 視圖 + 右側設計說明。底部為原始平面圖縮圖，可隨時對照。

**簽名元素**：
1. 平面圖到 3D 的過渡動畫（2D 線條「抬起」成為 3D 牆面）
2. 設計對比滑桿（Before/After 效果）

**互動哲學**：像翻閱一份精美的設計提案書，每個空間都有清晰的設計說明與視覺呈現。

**動畫**：頁面切換時的平滑過渡、3D 相機的慣性運動、hover 時家具的微微發光高亮。

**字體系統**：Noto Sans TC（中文）+ Space Grotesk（英文），現代幾何感與中文的搭配。

## 選定方案：方案三 — 互動式設計提案

選擇理由：最符合用戶「嚴格基於平面圖」的需求，2D/3D 無縫切換能讓用戶清楚看到設計與原圖的對應關係。分頁式結構也便於逐一說明各空間的設計建議。
